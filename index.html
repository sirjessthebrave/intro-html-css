The Internet and the World Wide Web

The Internet

Network of networks that connects millions of computers and servers gobally.

Information is passed around to connected computers via a veriety of languages called protocols for example HTTP (Hyper Text Transfer Protocol) or FTP(File Transfer Protocol).

The World Wide Web

A way of accessing information from the internet. The WWW is a portion of the Internet. The WWW uses protocols called HTTP or Hyper Text Transfer Protocols. HTTP allows not only the transfer of data, but also the ability to communicate buiness logic/functionality. Common HTTP protocols are: Get, Post, Delete, and more.

What is Web Development?

Web Development is the action of creating code that is hosted and connected to the Internet/World Wide Web. Making websites, web apps, web services, or any other peice of functionality that lives on a server connected to the web would be considered web development.

Web Development often falls into two categories:

  • Front End Development
  • Back End Development

Front End Web Development

Front End Web Development refers to Client Side code, or code that clients (users) interact with. For Example: The construction of as well as look and feel of a web page or application.

Back End Web Development

Back End Web Development is the creation of code that lives on a server which runs the logic/processes of the web page or application. This refers to Server Side code, or code that the server runs. For Example: If you site has users, the back end code processes and stores user data which is created, updated, or deleted and retured to the client/front end of your site or application

CSS

CSS stands for Cascading Style Sheets

CSS controls the look and feel of your site

Using an exterally linked peice of code you can change the colors, fonts, some layout, and whitespace of your site! Lets see what some sites look like without CSS.

Classes

Classes are elements you can include in your HTML to reference in CSS to apply styles to those elements. You can reuse classes over and over again.

  • Class
  • 
            //reference classes 
            with a period before 
            .blue-section {
              background: blue;
            }

Id's

ID/s are also elements you can include in your HTML to reference in CSS to apply styles to those elements, however you can only use ID's ONCE.

  • ID
  • 
            //reference classes 
            with a hash before 
            #blue-section {
              background: blue;
            }

Making space

Padding and Margin are ways to add space around your html elements. Padding adds space from the outside of an element and pushes items in, Margin adds space around the outside of your element and pushes out.

Including our CSS

Cascading

The styles in your 'Cascading Style Sheets' are exactly that - cascading. That means that there is a heierarchy of which styles take precidence. The styles on the bottom will take precidence to the ones that come before

However - if your styles are more specific they will override anything else!
